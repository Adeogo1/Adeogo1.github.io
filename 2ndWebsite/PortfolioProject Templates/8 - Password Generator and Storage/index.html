Password Generator and Storage

A web based application that creates a passowrd based on the application and stores it on the cloud for easy lookup

  • Intro
  • Work

Intro

A web-based application that generates and stores passwords for various sites and applications on the cloud. The user can create a unique password for each account using the application and then store it on the cloud for easy access. The application uses a secure encryption method to protect the passwords and can only be accessed with a master password set by the user. This feature ensures that the passwords are safe and can only be accessed by the user. The user can also update or delete the passwords as needed. This application makes it easy for users to have strong and unique passwords for all their accounts without having to remember them all. It also eliminates the need to write down or store passwords in unsecured locations.

This application helps to keep all your accounts safe by ensuring that a hacker could not gain access to multiple accounts with just one password. With this application, you can easily generate strong and unique passwords for each of your online accounts, and have the peace of mind that your personal information and online presence are protected. Additionally, the cloud-based storage allows you to access your passwords from anywhere, making it convenient for you to log into your accounts from multiple devices. This ensures that your accounts are always protected and your personal information remains secure.

Work

The program prompts the user for the length of the password, the number of symbols to include in the password and the number of numbers to include in the password using the Scanner class. These values are stored in the variables passwordLength, numSymbols and numNumbers respectively.

A secure random number generator is created using the SecureRandom class

A string called allChars is created which contains all the possible characters that could be included in the password. This includes lowercase and uppercase letters, numbers and symbols.

A variable password is created as an empty string and will be used to store the generated password.

A for loop is used to add a random character from the allChars string to the password variable for a number of times equal to the difference between the password length and the number of symbols and numbers specified by the user.

Another for loop is used to add a random symbol from the allChars string to the password variable for the number of times specified by the user. This loop uses the Character class to check if the character is a letter or digit and if so, decrements the loop counter and continues to the next iteration.

Another for loop is used to add a random number between 0-9 to the password variable for the number of times specified by the user.

The password is then shuffled using another for loop and a random index is selected from the password string. This is done to make the password more secure as it makes it harder for someone to guess the password if it is in a predictable order.

The shuffled password is then stored on the cloud using a storeOnCloud method.
