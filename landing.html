LEAP.LEARN.LIVE!

LEAP.LEARN.LIVE is a 2D endless platformer where you control a bouncing ball to move left and right, dodge obstacles, and avoid falling to your death. You can also collect coins which can be used to buy items in the store. This was created on unity using C#. This games uses unity’s touch control and buttons to create the effect the when the player touches the left or right side they move in the corresponding direction. It also takes advantage of unity’s data perception to save the money and uses it to buy items that can be used in game .

Around The World

Around The World is a 2D endless runner as well, where to begin you choose whether to go straight in or start a mini tutorial your difficulty which is decided by the map environment, Desert being easy, Jungle being medium and Arctic being hard and each of those maps have different obstacles some high and low but the player’s ability to double jump allows him to get past them well enough. There are also pickup that spawn randomly at different points in the game. The abilities are : Triple Jump – player gets an extra jump for the duration of the timer at the bottom and the colour of the UI depends on the map e.g. the arctic has blue UI, The next one is Increased Jump – This increases the player’s jump height by 50% which is quite a lot, the next one is shrink – this shrinks the player whole body size by 50% as well but everything else stays the same (excluding the collider) and the last one is invincibility – which allows the player to not get killed by any obstacle but they can still die if they fall in the water. The UI also changes based on the map using color theory I learned in order to have UI that looks nice together and doesn’t stick out like a sore thumb and I added some UI effects for example in the desert when you start you see a sandstorm blow towards you and when you die the UI also has a sandstorm, this is still the same thing with the Arctic but with snow instead that lands on the floor. This also the same with the river for each map. The settings menu also consists of a volume slider which control the background volume in the game because the Desert has wind and water waves, the Jungle has jungle noises and water waves and the Artic has water waves. The game also has gems which you can collect and use in the store to buy some skins unfortunately I couldn’t find any free assets to use .

Count

This project was based on shooting a basketball in to a hoop and counting how many times the ball went in. This project required a good knowledge of maths and the kinematic equation as you couldn’t simply place a rigidbody on the object and expect it to work exactly. The project also uses that data to create a line predicting the trajectory of the ball as you can see in the image.
